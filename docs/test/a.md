你可以使用自定义容器来突出显示特定信息。

::: tip 提示
这是一个提示容器，非常适合用来分享小技巧。
:::

::: warning 警告
这是一个警告容器，用来提醒用户注意潜在的问题。
:::

::: danger 危险
这是一个危险容器，用于警示用户某些操作可能会带来风险。
